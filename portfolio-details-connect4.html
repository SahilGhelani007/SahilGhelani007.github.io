Overview

Developed with teammate Yaw Nimo-Agyare for the Introduction to Artificial Intelligence course under Prof. Shivanjali Khare. Implemented an AI agent that plays Connect-4 optimally using the Minimax algorithm with Alpha-Beta pruning and a domain-specific heuristic evaluation.

Highlights

  • Implemented Minimax + Alpha-Beta pruning to efficiently explore the game tree.
  • Designed a heuristic evaluation scoring center control, threats, and blocking opportunities.
  • Built a Python UI to play vs. human or automated players; included difficulty levels via depth limits.
  • Gained hands-on exposure to adversarial search and strategic AI modeling.
