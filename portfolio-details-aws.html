Real-Time Website Traffic Analytics Using Distributed & Scalable AWS Services

Jan 2025 – May 2025 · University of New Haven

AWS Kinesis Lambda S3 Glue Athena Power BI Data Engineering Serverless

Overview

Built a cloud-native, real-time data pipeline to simulate and analyze web traffic using NASA’s HTTP Web Server Logs. The pipeline ingests, processes, stores, and queries high-velocity logs using AWS Kinesis, Lambda, S3, Glue, Athena, and visualizes insights in Microsoft Power BI—surfacing traffic trends, top URLs, and system health metrics.

Architecture

  • Ingestion: Log producers → Kinesis Data Stream
  • Processing: AWS Lambda for cleansing/enrichment → partitioned data in Amazon S3
  • Metadata & Query: AWS Glue Data Catalog + Amazon Athena for serverless SQL
  • BI Layer: Power BI dashboards with scheduled refresh

Highlights

  • End-to-end serverless design with scalable ingestion and fault tolerance.
  • Modeled partitions/bucketing to optimize Athena query performance and cost.
  • Delivered KPI views: Requests over Time, Top URLs, HTTP Status Distribution, Geo View.
